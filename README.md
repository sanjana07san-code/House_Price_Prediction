## Project Overview

The House Price Prediction System is a Machine Learning and Flask-based web application that predicts residential property prices using various housing features such as location, house area, bedrooms, bathrooms, parking spaces, floors, and house age.

The application helps users estimate property values and demonstrates the practical implementation of Machine Learning in the real estate domain.

## How to Run

### 1. Install Required Packages

```bash

```

### 2. Run the Flask Application

```bash
python app.py
```

### 3. Open Your Browser

```text
http://127.0.0.1:5000
```
